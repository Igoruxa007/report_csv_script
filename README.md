# report_csv_scipt
Для использования скачать файл main.py и папку scripts  
Запустить командой:  
`python main.py path_to_file_1 path_to_file_2 --report payout`  
- 'path_to_file_1 path_to_file_2' - пути до файлов с данными в формате csv через пробел  
- '--report' - тип отчёта
